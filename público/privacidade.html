Política de Privacidade

Última atualização: 02 de maio de 2026

A Psi Real ("nós", "nosso") está comprometida com a proteção dos seus dados pessoais. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários do sistema Psi Real.

1. Dados que coletamos

Coletamos os seguintes dados para o funcionamento do sistema:

  • Nome completo e e-mail para criação de conta
  • Dados de pacientes inseridos pelo próprio usuário (psicólogo)
  • Informações de agenda, sessões e registros financeiros
  • Dados de acesso ao Google Calendar, mediante autorização expressa do usuário

2. Como usamos seus dados

Os dados coletados são utilizados exclusivamente para:

  • Funcionamento do sistema de gestão clínica
  • Sincronização com o Google Calendar, quando autorizado
  • Comunicações relacionadas ao serviço
  • Melhoria contínua da plataforma

3. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto quando necessário para o funcionamento do serviço (ex: Google Calendar API) ou por obrigação legal.

4. Google Calendar

Quando você autoriza a integração com o Google Calendar, acessamos apenas os dados necessários para criar, editar e excluir eventos de sessões. Não acessamos outros dados da sua conta Google. Você pode revogar esse acesso a qualquer momento nas configurações do sistema.

5. Segurança

Utilizamos criptografia e boas práticas de segurança para proteger seus dados. As informações são armazenadas em servidores seguros via Supabase.

6. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou incorretos
  • Solicitar a exclusão dos seus dados
  • Revogar consentimentos dados
